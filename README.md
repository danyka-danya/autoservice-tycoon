# 🔧 Автосервис: Империя

Браузерная idle/tycoon-игра: путь от полуразвалившегося гаража до международной сети автосервисов стоимостью $10 000 000.

## Как запустить

1. Откройте папку `autoservice-tycoon`
2. Дважды кликните файл **index.html**
3. Всё! Игра работает целиком в браузере, интернет не нужен.

Прогресс сохраняется автоматически (LocalStorage браузера). Не чистите данные сайтов — иначе сотрётся сохранение. На всякий случай есть Экспорт/Импорт в Настройках.

## Управление

- **Пробел** — пауза, **1 / 2 / 3** — скорость ×1 / ×2 / ×4
- 1 секунда реального времени = 1 игровая минута
- Ночью (когда сервис пуст) время летит в 18 раз быстрее

## Структура проекта

```
index.html          — оболочка и порядок загрузки модулей
css/style.css       — дизайн-система (тёмная/светлая тема, 5 акцентов)
js/core/            — утилиты, звук, сохранения
js/data/            — весь контент (машины, поломки, детали, миссии…)
js/game/            — игровая логика (экономика, ремонт, персонал…)
js/ui/              — интерфейс (сцена, панели, модалки, графики)
js/main.js          — точка входа
assets/audio/       — сюда можно положить свои звуки (см. README внутри)
```

Модули подключаются классическими `<script>`-тегами в общий неймспейс `AST` — поэтому игра запускается двойным кликом по index.html без веб-сервера.
