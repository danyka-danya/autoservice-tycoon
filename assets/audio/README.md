# Свои звуки для игры

По умолчанию все звуки синтезируются автоматически (WebAudio) — файлы не нужны.

Хотите свои? Положите mp3-файлы с такими именами прямо в эту папку — игра подхватит их сама:

| Файл | Когда играет |
|---|---|
| `music.mp3` | фоновая музыка (зациклена) |
| `cash.mp3` | оплата ремонта |
| `click.mp3` | нажатия кнопок |
| `buy.mp3` | покупки |
| `error.mp3` | ошибка / не хватает денег |
| `level.mp3` | повышение уровня |
| `car_in.mp3` | клиент приехал |
| `car_out.mp3` | машина уезжает |
| `repair.mp3` | процесс ремонта |
| `wrench.mp3` | завершение работы |
| `achievement.mp3` | достижение |
| `chest.mp3` | сундук / награда |
| `event.mp3` | случайное событие |
| `notify.mp3` | доставка и уведомления |

Включение/выключение звука и музыки — в игре, вкладка «Настройки».
